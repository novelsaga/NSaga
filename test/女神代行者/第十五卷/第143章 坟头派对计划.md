# 第143章 坟头派对计划

死魂灵，亡灵系暗兽，根据神话传说中的记载，大概率为墓场女神的常用护卫。
本质上，就是一个凝结了许多怨灵的浮空球体。
其体形在描述中差距巨大，从直径三米到直径十米皆有传闻。
不过已经知道对方属性的薛雷很快就明白，那是对方特性发动的结果。
亚克埃尼米不仅在神话中的地位高出海洋那些女神一头，护卫的特性，都高达六个。
——无魂者，可以让死魂灵不受任何精神伤害。
——怨灵尖啸，每隔一段时间，就会以死魂灵为中心扩散出猛烈的精神伤害，范围会根据体形的不同而大幅变化。
——灵体化，与蒂尔宁的绝活一致，没什么好说的。
——聚合，可以短时间内将体型压缩至直径半米，来大幅提升魔能。
——幽骨召唤，能随机施放亡者相关的召唤术，因为并非死灵法术，不需要尸体或残骸。
——遗愿，将其击杀的个体将受到所有怨灵的诅咒，持续至被抗性消磨为止。
——灵能榨取，暂时吸收附近弱小生物的灵魂增强实力，恢复伤害。
在此期间被击杀，则所有灵魂都会一起湮灭。
论战斗经验，薛雷是纯正菜鸟，古莎是新瓜蛋子，而蒂尔宁，是典型的和平万岁主义者，最擅长各类自保技能和白魔法，指望他们三个商量出一套行之有效的方案，不如掏出火晶石求助今天新入队的同伴。
露狄小心翼翼坐在桌边的椅子上——她对于木制家具有种很合情理的谨慎，毕竟情绪一个波动，屁股下面就没东西可用了。
“请看看，这是此次作战要面对的第一个难关，我想知道，你有什么很好的解决办法吗？”薛雷把纸放在女炎魔面前，伸手挡在她的嘴和纸之间。
露狄有点羞恼，瞪着他说：“我不生气的时候嘴里不会喷火！”
他笑着摇摇头，“可你看起来就一副总是在生气的样子。露狄，其实你不生气的时候很漂亮，多笑笑就更可爱了，不要老是这么紧张兮兮的。”
她没有笑，反而把脸板得更厉害，盯着纸上大串的句子，不吭声了。
〖人家估计把你当强奸犯了。
身上就嘴巴一个洞，还被你硬上，怎么笑得出来嘛。
〗
薛雷没回心音，反正强奸这事儿，他穿越第一天就在琳琳身上试过了，一回生二回熟，虱子多了不痒债多了不愁，随意吧。
可等了一会儿，露狄的表情竟然更加凝重，如果不是人类外形只是化身，保不准都要冒出一层冷汗来。
“这个暗兽很难对付吗？”薛雷也跟着心情沉重了几分，如果连护卫都对付不了，那还怎么去找出隐藏在广阔神域中的亚克埃尼米。
露狄绷着脸，小声说：“我不认识这些。”
“啊？”
她抬起脸，阴郁地说：“我，不认识，这些，单词。”
“呃……我专门用你们魔堡这边的惯用方言写的啊。而且还做了冷僻词汇的注释。上古专属名词我都替换成了现在比较好理解的内容，具体你哪里……”
〖雷哥，闭嘴吧。
人家不认字。
咱们来自一个几乎没什么文盲的地方，可不代表这边也有九年义务教育。
〗
露狄的表情果然更加困窘，要不是嘴巴紧紧闭着，可能牙缝又要磨出火星。
古莎笑嘻嘻在旁说：“吾主，炎魔的女孩子出生后就知道打架，打来打去的，没功夫学拼写啦。还不如我们魅魔有文化呢，我妈妈那时候逼着我学认路牌，说是找食物能容易一些。炎魔都没有这个需求，就是你们学院招生时候说的那个……那个什么词来着？文……”
薛雷直接伸手，把文盲单词的后半截捂进了嘴里。
倒不是他对异世界的环境缺乏共情。
如果露狄是个下层平民，一辈子要为了生存挣扎，没有空闲学习再自然不过，只能靠丰收女神的赐福来逐渐改变。
可寿命漫长的高等魔族，不是有很充裕的时间去吸收各种知识吗？
光明侧的长生种可是几种语言换着说都毫无问题的。
他把纸拿回来，柔声说：“不要紧，我给你念。露狄，如果今后你对通用语这方面的知识有兴趣，我可以抽空教你。”
她抬起眼，带着刺反问：“我为什么要学那种没用的东西？能帮我变得更强吗？书本用来焚烧都不如木头，是人类最愚蠢的玩具。”
“力量武装你的身躯，而智慧可以武装你的头脑。你父亲比你聪明得多，我猜，他应该能看懂这些句子吧？”
露狄咬了咬牙，“女炎魔没有那种时间。除了练习战斗，我们还要为漫长的生育做准备。”
“从今以后你有时间了。你父亲把你赠给了我，我更愿意你读书识字，学会吸收知识，学会体验各种战斗之外的快乐。”
“你要先完成任务。否则，即使有血魂契约，我的同族也会击杀你来拯救我。除了魅魔，魔族的女性都是宝贵的财富。”
古莎撇撇嘴角，“我们是人类男性的梦想，才不稀罕和你们一样成年就要准备生孩子。”
无意也没有时间深入讨论异世界社会问题，薛雷摆摆手，说：“好了，先回到正题。露狄，我会给你念一下内容，你仔细听着。”
等到念完，露狄板着脸说：“我为什么要相信你这不明来源的情报？幻兽、暗兽之类的知识，在最老的恶魔口中也没有提到过几次。你是怎么连死魂灵的特性都弄清楚的？”
薛雷笑了笑，手指轻轻点了点额头，“因为我是女神的代行者，薇尔思是创世女神之一，她被背叛的时候还没有后来这些伪神，但这些伪神召唤的护卫，用的力量有一部分正属于薇尔思，她可以从中了解很多，并传达给我。”
“神谕吗？”
“差不多就是那个意思。”
“神谕那种东西可信吗？”露狄的语调陡然拔高，“占卜师中全是骗子，全部烧死也没几个冤枉的，神官不过是高级一些的占卜师，你让我依靠这种情报来安排战术？”
“这不是需要神官凭自己主观理解来翻译的那种模糊谕示。这是女神直接赐予我的启示。我是代行者，不是神官。”薛雷充满自信地说，“你只需要把这信息当成绝对无误的情报就可以，如果出了错，我负责。”
〖哇，雷哥忽然好有男人味啊。你打算怎么负责？〗
带她跑，让她父亲惩罚不到她。
〖果然……〗
露狄双手在大腿之间扶着凳子，苦苦思索了片刻，说：“我可能打不过这种怪物……你的魅魔女仆，如果用暗系魔法的话，胜算也不大。父亲派来的魔兽都是可驯化的品种，好用但是很弱，怨灵尖啸的威力如果够强，它们撑不了几次。你这只麦伦的实力如何？”
被女炎魔的眸子锁定，一直卧在旁边默默吃蘑菇的蒂尔宁眨眨小鹿一样无辜的大眼，小声说：“我才进化了两年多一点，前一阵子刚刚有自己的领地，就被这个人类用美食拐跑了。我觉得我很弱。”
可怜她那点面对人类男性的小小骄傲，最近被反复碾压，已经不敢再轻易表露出来了。
不过私下和薛雷相处的时候，她还能重拾一点自信。
毕竟，魔法免疫的他最需要的就是她这样的帮手。
重要性，从来都是女孩子的底气。
哪怕只有半个身子是女人，也不例外。
“我关心的，是你的白魔法造诣。”露狄不耐烦地说，“我不关心你的年龄，古莎比她妈妈强好几倍。”
蒂尔宁想了想，大致交了个底。
她可以晃晃尾巴跟着骑手过吃得香睡得更香的美好生活，但同胞们还要在苍穹魔堡生活，得罪炎魔的代价可比得罪魅魔大多了。
“嗯……一只精英麦伦，几十只可以临时帮忙顶一下战线的蠢鹿，死魂灵的强度不超过我父亲的话，这一战还有得打。而且暗兽的资料这么隐秘，大家都不知道，揍扁它也有理由可找。我看看能不能从爸爸那儿借点好打手。”
薛雷皱起眉，“我不在乎你们魔族那边你打算怎么安排，但跟随我的古林灵鹿，我不会让她们中的任何一个为了临时顶住战线而牺牲。你最好不要做这种计划。”
“愚蠢的人类，这是和暗神的战斗，说是用性命的赌博也不为过。参战的所有个体，都要有没命的觉悟。我如果出手，也一定做好了化为火星彻底消散的准备。同情心泛滥的家伙，不要试图指挥这种战争！”
薛雷当然知道慈不掌兵的道理。
可他又不是什么将军，这也不是他非要牺牲一切拿下的战场。
以他的私心，真要堆炮灰，当然还是毫无关系的下层魔族们来负责更好。
露狄显然也不是实心木疙瘩类型的蠢货，端详了一下他阴沉的表情，就闭着嘴磨了磨牙，说：“我知道你的意思，我只能承诺，不到万不得已，保证不让那群鹿去送死。还是说，你打算告诉我，她们的命比你的还重要？”
薛雷不会撒那种无意义的谎。
别说是蒂尔宁和她的同族小鹿，就是全部古林灵鹿加到一起，也不如他重要。
这不光是代行者的重任，也是一个正常人的私心。
但事情本来就不只有非黑即白的二择，他清清嗓子，说：“如果真到了需要牺牲小鹿们来拖延时间的地步，那么，我选择撤退。保存有生力量，谋求下次的胜利。”
赶在露狄再说什么之前，他抬手打断，“我和女神的战斗方式，和你想象的不同。我不可能在死伤无数的战场上完成那个过程。”
跟着，他很严肃地讲述了自己驱逐女神的方式。
露狄目瞪口呆，几颗火星在洁白的门牙之间来回跳跃。
蒂尔宁则迅速调整了卧伏鹿身的朝向，让浑圆白嫩的诱人小屁股远离了这个可以当众凌辱女神的强者。
“好……好吧。”不愧是炎魔一族的精英血脉，露狄先一步调整好了情绪，“我……我会把作战方式调整一下，不以歼灭为主，重点是，给你制造……制造……制造……让……亚克埃尼米受精的……机会。”
女炎魔用力拉扯了一下暗红色的短发，烦躁地说：“所以我的任务就是在你找出来女神并肏飞她之前拖住所有阻碍的力量，对吧？”
“对。”薛雷很淡定地回答，“之后暗神复苏的每一次，我都要来重复这个过程。这边是你父亲的领地，我带太多帮手不合适，你正好演练一下。毕竟，没意外的话，暗神每次复苏，都会比上一次强一点点。”
“那岂不是最后还会完蛋？”露狄皱起眉，眼神流露出明显的凶狠。
这让他觉得，自己往身边养了一只真会吃人的母狮子。
“不会，驱逐够一定次数后，我就有信心收服她。到时候她就不会再有现在的威胁能力。魔堡也就安全了。”
她沉默着思考了一会儿，小声咕哝说：“我本来该觉得你这些话都是他妈的胡扯。”
“可是呢？”薛雷笑了笑，追问了一句。
“没有什么可是。”露狄隐藏了真正的理由，舔了舔牙齿，“既然我父亲将我当成报酬，我没道理不好好帮你。如果你死了，我就不用再受血魂契约牵制，如果你说的是真的，那么，我将得到进化的机会。这对我有什么坏处？来吧，我来制定作战计划，我们来做你的床帐，让你在里面，安安全全地肏公主。”
〖雷哥，坟头蹦迪就够狠的了，你这是准备坟头开性爱派对啊。〗
是啊，对付那种名字就死气沉沉的暗神，就是要好好热闹一下才行。
不然画风都要一转变去恐怖片了。
魔族可以不管村庄居民的死活，薛雷不能不管。
就算这些人不是他未来的信仰来源，他的同情心也不允许他眼睁睁看着这么多无辜民众被波及卷入，损失惨重。
一只炎魔和一只魅魔，两位魔族精英表现出服帖态度的情况下，他的话也相当好用。
憨厚老实的中年女村长表现出了十足的敬意，身边带着的黑发小女孩也看起来恭顺而有礼。
可惜蒂尔宁似乎不喜欢这个性别的人类幼崽，直觉上就感到厌恶，远远躲开了。
女村长的人生大概不是很顺遂，饱经岁月折磨的脸上充满了疲态，口齿都不是太清楚，交流的对象，不知不觉就换成了那个即将长成为少女的小姑娘——吉维萨。
“尊贵的先生，我无意探寻您和两位贵族的秘密。”她虽然看起来很胆怯，但说话的时候，依然直视着薛雷的眼睛，“我只是很好奇，在这么靠近魔堡的腹地，为什么会出现可能被战斗波及的危机？这村子虽然不大，可我们经营这个小小家园，也付出了许多努力。”
“我可以补偿你们足够的财产，等到事情平息，你们也可以立刻回来。这地方虽然离魔堡不远，但我看你们生活得挺艰难。”
她小幅度地摇了摇头，“先生，财富的补偿能说服大部分人，但总会有比较固执的长辈，需要我和妈妈给出一个无法拒绝的理由。”
露狄呲了呲牙，“我还不算是很好的理由吗？”
吉维萨瑟缩了一下，小声说：“您是个理由，但并不好。有些愚钝的臣民会误认为您打算强夺大家的土地。”
为了让人类给魔族的地盘创造足够丰富的物质基础，领主们大都遵守着多年摸索出来的规则。
对这个寿命短暂实力孱弱但创造力、潜力和承受力都无比强大的族群，给他们一块安稳居住的土地不去打扰太多就能爆发出无限可能。
所以才会有低阶魔族出现在人类村庄附近担任基础守卫工作。
据说人类最不愿意被夺走的东西除了生命之外，就是家园。
露狄犹豫了一下，看向薛雷。
薛雷露出和善的微笑，“小妹妹，我知道你的担忧。我们要和一个非常强大的对手作战，谁也不知道到底会发生什么，让你们暂时搬迁，是为了保住你们的命。”
“可是，有领主派部下暗中告诉过我们，名为薇尔思的女神会庇佑大家的啊。”吉维萨的语调带着天真的期待。
薛雷的笑容更加和气，心里决定下次见到夏拉肏她的时候要温柔些，“对，我就是那位女神的代行者，我正是为了庇佑你们不被与邪神的战争波及而来。”
吉维萨小小的脸上顿时露出了放心的神情，“那真是太好了。我和妈妈今晚就挨家挨户走访，一定保证说服所有人明早搬家。可是……”
她稍微有些为难地说，“村庄的财产不多，大家可以带上。但周围的荒野非常危险，村里又没有可靠强壮的叔叔。代行者先生，会有人保护我们吗？”
薛雷看向露狄，露狄摊摊手，模仿人类惯用的姿势表示无能为力。
上次一匆忙，忘了给夏拉塞个灵魂结晶，真是失策。
虽说让古莎临时飞过去也来得及，但此刻已经身处亚克埃尼米的广阔神域覆盖之中，他不敢掉以轻心放走最可靠的保镖。
蒂尔宁犹豫一下，开口说：“我想，从这里往魔堡转移，只走大路的话，还是很安全的。本来这个方向有个新空港要筹建，沿途的魔兽巢穴大都被强制搬迁了。”
吉维萨松了口气，“那就好。啊，对了，代行者先生，明天会很早就开战吗？”
“还不知道。你问这个干什么？”
“还不知道什么时候能回来，我离开前，想去给爸爸的坟墓换一束鲜花。”
“呃……”虽然女孩浮现出淡淡哀伤的大眼睛特别动人，但薛雷总不能让这么一个楚楚可怜的小姑娘跑去暗神最可能的藏身地找死，“那种事情不是很重要吧，我建议你还是等回来再说。”
“可是……后天就是爸爸的忌日。我和妈妈，明晚就能回来吗？”
“嘶……”薛雷为难地挠了挠头，“这个还真不清楚，毕竟对手还藏着呢，什么时候能找出来，不好说。”
露狄压低声音说：“这不是很好的试探机会吗？让村民走前都去坟地祭拜一下祖先长辈，说不定还能挖出这里潜伏的信徒。”
“信徒？”
“如果这里没有大量信徒，暗神为什么会选择在附近藏身？”露狄差点控制不住音量，“我甚至怀疑这小女孩的妈妈就是信徒之一，你是人类你感觉不到，她身上弥漫着一股死亡的气息。”
咦？真的吗？薇尔思，我接触一下她的话，你能不能鉴别出来？
【不行。在这个伪神的领域内，我的很多能力都被压制了。所以我才提醒你，一定要格外小心。她的权能在露比娅的女儿中大概也是最强那一档次，我毕竟不是本体亲临……】
担心露狄是在找借口安排人类送死当炮灰，薛雷拉过古莎咬耳朵问了问，这才确信。
魅魔公主也承认，村长的身上有一股“不祥”的味道。
“不过这个村子就位于亚克埃尼米的神域之中，沾染到气息也很正常吧？”
古莎摇摇头，小声说：“吉维萨就没有。她的气味就很……唔……很纯粹。”
“纯粹？”薛雷不解，“这是个什么形容？”
“就是没有融合异质的那种味道。比如，吾主你是薇尔思殿下的代行者，身上就会有薇尔思殿下的味道。正常是感觉不出来的，跟吉维萨那种纯粹的味道来比较的话，才能感觉出来。我说的味道不是你用鼻子闻的那种，反正……就是魔族比较爱用的说法。你可以把这这个理解为灵魂的颜色之类比较文艺的东西啦。”
既然如此，二五仔的确应该早点捉出来。
苍穹魔堡是和光明侧最友好的黑暗侧势力之一，薛雷还希望在这边建立稳固的信仰据点呢。
“好吧，那这样，明天早晨你们所有村民出发前，都可以先去坟地祭拜一下亲人。我会和同伴一起保护你们。好吗？”
吉维萨露出灿烂的笑容，“好的，谢谢先生，先生这样的代行者，背后一定有一个温柔又善良的完美女神。今后，我会向大家努力宣传薇尔思殿下的。”
“那我就先谢谢你了。”
