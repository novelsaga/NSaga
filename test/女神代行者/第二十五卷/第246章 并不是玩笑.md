# 第246章 并不是玩笑

经常有人说，男人偶尔会用下半身思考。那根到了发育期就会蠢蠢欲动一直间歇性饥渴到几十年后的生殖器，也常常被称为“小头”。
那大都是针对男性容易被兽欲支配而做出的调侃。
但这次，薛雷在慢慢玩弄莎的屁眼时，真有了一种自己的小头也开始学着思考的错觉。
都说苦难能够磨砺心智，看来最近密集的直肠之旅，让小兄弟的智慧也有了长足的进步。
他揉搓着莎白晰柔软的臀肉，想通了一件对他来说可能非常重要的事——信仰和权力，并不是非要全部捆绑在他一个人手中。
这个看似不搭边的结果，确实有明晰的思路。
薛雷先是在想，应该怎么样才能让莎放开一切对他完全臣服。然后意识到，他俩的主要阻碍在于不能稳定地见面。
他想给莎一个稳定幽会的借口，于是想起了库里提卡卡塔那边对创神会的邀请。
那么，让莎担任教会中的重要职务，教宗过去视察的时候，难道还能没有私下见面的机会？
但总是指名叫一个普通祭司跟教宗见面，莎的丈夫又不是傻子，就算作为赘婿没有意见也变不成歪嘴龙王，对库里提家的名声，多少是个负面影响。
他就想，反正莎这个样子看起来不是什么很难对付的角色，要不干脆库里提卡卡塔的教会管理者，就任命给她得了。
这个念头一出现，他此前一直在发愁的事情，就豁然开朗，打开了全新的局面。
毫无疑问，现在正是创神会在洛库尔兰大举传播的最好时机。
而最大的阻力，除了军方派系不愿意松口之外，还有可靠的管理人员数量不足，捉襟见肘的问题。
人鱼之冠那边正在抓紧统合整个水鬼海湾，派往希拉玛卡塔的队伍，几乎就是当前能压榨出的全部余力。
假如舞会结束之后王国元帅法瑟长公主忽然抽风松口，给予创神会全境传教许可，薛雷都不知道该怎么分配人手去覆盖一个如此辽阔的王国。
他头疼苦恼，觉得无计可施，最主要的原因，就在于他脑子里僵化的坚持。
他恨不得开设在各大领地的教会全部由最虔诚的信徒掌控，最好都是穷苦百姓出身，实在不行，再考虑契约了信仰的下层贵族。
而在决定让莎成为自家领地教会管理者以便将来幽会之前，薛雷必须要好好分析其中的利弊。
仔细思考之后，他猛然发现，如果他不执着于所谓的掌控权，放手交给莎这样的当地贵族去做，显然对信仰的快速传播更加有利。
他一直不愿意让自己一手创立的教会成为看过的各种故事中典型的反派——为统治阶层谋取利益维护权力的工具。
但肏着莎温暖紧凑的屁眼，凑巧抽离为比较客观的视角后，他不得不承认，把利益捆绑到一起，交给各地贵族去推动，才是传播最快最有效率的方式。
至于他本人可能因此而损失的权威，其实并不重要。
因为他背后的女神并不是虚构的传说，而是真切存在的伟力。信仰高速增长能反哺薇尔思的恢复，他这个代行者，也将因此而更将强大。
他不是君权神授之后就会被国王架空，最后守在私有领土里当吉祥物的那种教宗。
如果这世界上到处都是信仰薇尔思的虔诚教徒，他才能获得最大的靠山。
真实神明存在的世界里，他直到此刻才醒觉这个简单的事实，可见曾经在无神世界生活的时光留下了多么深刻的影响。
他不需要紧紧握住权杖。
他，即是权柄自身。
听到他的提议，莎裸着身子坐在那儿愣了好一会儿神，才小声说：“教宗先生，你是在开玩笑，对吗？”
“当然不。”
她的嘴唇颤动了一下，“可是，你知道的，我只是个用来生育后代的……不成器的女儿。父亲真正器重的，是我其他兄弟姐妹。”
“这不重要。”
薛雷微笑着说，“一切顺利的话，今后冠军领主大人一定会非常器重你。远超其余兄弟姐妹。至于你之前的职责……别忘记，我是丰产女神的代行者，再没有比甘愿奉献身体来繁衍族群更伟大的行为了。”
莎的呼吸急促了一些。她并不笨，她当然知道，薛雷的承诺如果兑现意味着什么。
“那，代价是什么？”
薛雷差点没忍住摆个古尔丹的造型。
他憋了憋，清清嗓子，本来打算说什么都不需要，但转念一想，在贵族家长大的女孩，越这么说她越不会相信，反而是有明确的利益交换，才能让对方感到安心。
他思考了一会儿，说：“你得受恩成为信徒，并比其他人更虔诚，愿意付出一切来侍奉女神……和她的代行者。”
莎抿紧唇瓣，坐在那儿蹙眉深思，连臀缝里垂流下来的黏液染湿了床单都没注意。
几十秒后，她轻声说：“我明白了。我一回去，就结束我的婚姻。我会做好准备……把一切都献给你。只是……你、你真的……会让我来管理库里提卡卡塔的创神会？我现在甚至还没有入教。这真的不会太草率吗？”
“不会。”薛雷含笑抚摸着她雪白柔软的脊背，意味深长地说，“这是我关于新决定的一个尝试。”
更多的内容，他不认为有必要跟莎深谈下去。
豁然开朗的局面之后，还有许多错综复杂的因素，为了将来教会不分裂成几个派系互相指责异端发动圣战，他得跟真正的专家——文拉尔好好商讨一下具体操作细节。
离开卧室，薛雷随便找了个空房间，由穆琳担当接线员兼声优，和文拉尔进行了一次长达四十多分钟的临时会谈。
结束之前，文拉尔的口吻变得有些微妙。
〖教宗，我真没想到，您会愿意如此。这确实是最有效率的做法，但也确实不像是您乐于看到的。您在洛库尔兰……没有遇到什么麻烦吧？〗
放心，没有。
他平静地回答，我也在不断学习，不断成长，想要更好地为了女神努力。
只要对薇尔思殿下更好，我个人的一些小小喜恶，不值一提。
对于契约了信仰的文拉尔来说，神恩主教的所谓实权，同样远不如女神的荣光传播效率重要。
她甚至不介意为了更快取得许可，设立一些教内地位和她齐平的职阶，方便游说洛库尔兰较为顽固的派系。
这一点薛雷暂时还不太情愿，只答应纳入考量。
他想法转变的起始，其实可以追溯到随口承诺将来让波赛思担任此地虔心者的时候。
那会儿他只是觉得，希拉米特家非常识时务，对创神会进入洛库尔兰地区可以说立下了汗马功劳，应该让出了大力的波赛思得到一些快感之外的实际甜头。
现在一切豁然开朗，他反而觉得，自己只给出虔心者的职位，好像有点吝啬。
说到底，那也就是比祭司和修士高出一级的定点传教士而已，真论实际能得到的好处，兴许还不如同级别负责巡游各地的传颂者——出远门传教总得报销旅费吧。
他估计了一下，想要让希拉米特家这种地方豪强领主在紫月症平息之后还真正上心传教，恐怕至少要给到大神官这种仅次于神恩主教的头衔，才显得有诚意。
可惜现在对方还没有提出要求，他来主动反而会让局面不利，只能等待良机。
结束治疗的时间比预想中晚，薛雷下楼的时候，会客厅又已经只剩下波赛思。
她穿着极其贴合身段的黑色晚礼服，带着不加掩饰的媚笑款款走近，抱住他的胳膊，用仿佛碰一下就会挤出鲜奶的饱满乳房贴住他，娇声说：“辛苦教宗了，治疗全部结束，舞会也十分圆满，走，去我那里，吃个庆功宴吧？”
薛雷点了点头，顺口问：“布莉妲回去了吗？”
“没。”波赛思不紧不慢地说，“舞会要结束了，我还想问问，你打算怎么安排布莉妲呢。”
他一怔，跟着笑了出来，“她是希拉米特家的女儿，为什么要问我安排的事。”
“她是你虔诚的小追随者，跟在你身边的时间也不短了。今晚这么值得高兴的日子，你……不准备好好‘安排’一下她吗？”
波赛思的意思已经非常直白。
那隔着衣服磨蹭着他胳膊、微微发硬、还有些濡湿的奶头，更是给这层暗示加上了别的意味。
她似乎打算老带新，来一个同场竞姬。
但薛雷还记着布莉妲的小小心愿。他挺喜欢这种类型的女孩，愿意在力所能及的范围内满足她。
“她陪着照料我这么久，我决定安排她今晚好好休息。后面几天如果有空，就给我当向导，带我好好逛逛希拉玛卡塔。”
波赛思咯咯笑了两声，“那，你今晚就只跟我一起庆祝？教宗，你那么威猛，我一个人，担心吃不消呀。”
薛雷捏了一把她扭来扭去的屁股，“好不容易治疗结束，跟你一起吃过饭，我就回去住处休息了。这阵子都没怎么陪我的同伴，有点对不起她们。”
“我又只有一顿饭的时间啊？”她故意娇嗔地说，“那我要吃不饱，怎么办？”
“我还打算跟你谈些事情，时间不会太短的。”
薛雷拉开马车的门，对里面明显重新精心打扮过的布莉妲说，“让车夫送你回去休息吧。明天再来找我。”
布莉妲忐忑地看向波赛思，从姐姐那边看到首肯，才松了口气，轻声应下，跟着马车一起离开。
波赛思并不是欲火中烧发骚上头的女人。
等私宴的精致饭菜上好，女仆退下，她就收起了之前轻浮放浪的作态，脱下一侧肩带，掏出一颗涨鼓鼓都起了青筋的乳房，一边往杯子里挤奶，一边说：“我说，您又有什么重要决定了吗？教宗。”
薛雷摆摆手，提醒她私下不必用敬称，“不算重要，我就是打算，派人去库里提卡卡塔帮忙抵抗紫月症之后，把那边的新信徒，都交给莎负责管理。想问问你，如果我给她受恩后提拔成神官，你会不会觉得我偏心？”
波赛思挤奶的动作突兀地停顿了一下，不过马上，她就继续让乳白的汁液喷洒在杯子里，语调也依旧平稳，“那怎么会。别看我认识你没多久，我有自信，比你不少女伴都了解你。你会专门来跟我说这件事，就说明，如果我也想要做神官，你不会拒绝。顶多……是让我用比较特殊的方法央求一下嘛。对不对？”
她最后捏了几下，用软布擦干红艳艳的乳头，换另一边继续挤榨，微笑着说：“我估计你真正关心的也不是所谓的偏心问题吧。那，我得先确认一下，你刚才说的意思，是不是相当于，这次派去库里提卡卡塔的都是传颂者，根基建立之后，就全部交给库里提家管理？”
“不。”
薛雷摇头否认，喝下一杯香气浓郁的火葡萄，凑过去在杯口上方嘬了一嘴鲜母乳，品尝片刻，满足地靠在椅背上，说，“我说的是，交给莎管理。”
波赛思点了点头，擦干这边的乳房，收回衣裳之中，思索了一会儿，说：“从教会的角度，莎不是个好人选。她来管理，和库里提家直接管理，没有什么本质上的区别。而且，你真的下定决心了吗？”
“嗯。”
薛雷连借口都已经找好，“在希拉玛卡塔之外的地方，紫月症正越演越烈，我想要让信仰更快速传播开来，遏制邪恶月神的侵蚀，就必须做出正确的抉择。这种时候，不能太考虑私利。毕竟，对我来说，每多死一个，就相当于少了一个潜在的追随者。”
波赛思好奇地打量着他，托腮眨了眨眼，说：“这段时间，你应该对洛库尔兰贵族的办事风格有所了解才对。这种权力你不趁着紫月症还在肆虐的时候抓在手里，将来，他们有得是办法，让你再也碰触不到。”
他笑了笑，平静地说：“我并不需要直接碰触所有的信徒。我也无意以代行者的身份去控制所有信徒的人生。我需要的，是让更多人了解薇尔思，了解丰产女神，让他们明白什么才是真正值得信仰的。我觉得，我决定这么让步之后，军方那边的阻力，会不会也能小一些？”
波赛思叹了口气，“薛雷，我们家和你的合作，已经表现出了相当的诚意。咱们两个之间的关系，应该也能让我说一些你可能不爱听的话了吧？”
“你说。贵族的事情你比我了解，我今晚和你谈，就是因为相信你的判断。”
她点点头，敬了杯酒，也喝了口自己的乳汁提香。
然后，她清清嗓子，说：“你是不是认为，那些掌握军队的公主、长公主们对你一直保持警惕的态度，是担心你通过宗教渗透进来，威胁到她们的权力？”
薛雷点点头。
“所以你认为，把教会的地方管理权交给各位领主的直系亲属，比如这些跟你发生过关系的女人。你就可以用代行者授予的权柄作为对内的借口，领主权力不受影响作为对外的借口，顺便表明自己对信仰之外的东西不感兴趣，没有野心，来让公主殿下们放心？”
薛雷点点头，“这有什么不对吗？”
“从一开始就错了。”
波赛思摇了摇头，“你呀，身边要么是精灵，要么是魔物，要么是魅魔，信得过的人类，不是狂热的神学家，就是出身低微的奴隶。我这样的女贵族，你又不肯相信……”
她用餐刀切下一块汤汁浸润的嫩牛肉，叉子压在上面，说：“你真以为军方的阻力，针对的是你这个人，和你手下新兴的教会吗？”
薛雷疑惑地皱起眉，“不然呢？”
“这是洛库尔兰，除非薇尔思殿下的职能是战争女神，不然，咱们创神会根本不可能得到士兵的忠诚。如果担心的威胁是那个方向，早就有专业的刺客成群结队来抹杀你这个人的存在了。”
波赛思抬起手，指着自己的鼻子，“能让王室军队掌控者感到不安的，肯定是我们这些大贵族啊。”
“诶？”
“各个家族之间的利益错综复杂，并不是说，我们都在洛库尔兰王国，就亲如兄弟姐妹，对王室无限忠诚。没有人会嫌自己手里的权力太多。假如有个没任何后患的机会出现，能让一家贵族取代现在王室的地位，你猜会发生什么？”
薛雷想了想，嘲弄地说：“我猜你们自己会先打起来。”
波赛思没有否认，摊开手笑着说：“所以最让王室不安的，就是大贵族们找到共同的利益，不再彼此争斗。现在是和平时期，和那帮丑陋的魔族已经很久没有发生过大规模战争，但真正逐步减少军事投入，进行产业改革的领地，暂时只有我们希拉玛卡塔。愿意接纳创神会，全力支持你们来驱逐紫月症的，也只有我们希拉玛卡塔。人和人之间都难以互相理解透彻，更别说牵涉这么多的大家族之间。”
她用手指沾了沾乳汁，在桌上画了一个奶白色的圆，“这里是尚武的洛库尔兰，每个库尔兰人都愿意追随强者。王室的本家，历史上已经更迭过三次。其中有两次，都是实力派领主们共同进退，直接造成的结果。所以，创神会信仰的传播，真正让王室忌惮的，就是给大贵族们提供了一个联合到一起的可能性。至于这联合的背后是不是由你这位教宗来主导，之后是不是打算成立一个维护女神权威的教国，都不重要。”
她抬起头，盯着薛雷，“在这片土地上，谁拥有更多的士兵更好的铠甲更厚重的城墙更犀利的魔炮，谁的命令，才更有效。”
薛雷咽下嘴里已经咀嚼到没味道的肉团，皱眉说：“所以我还按之前的方式来进行，反而更好？”
波赛思摇了摇头，“不，没有更好。我只是提醒你，两者并没有太大区别。你用创神会在各地吸收信众，让洛库尔兰拥有共同的信仰，和你让贵族们自己在领地发展教会，彼此间拥有共同的信仰，对她们的威胁，几乎一样。”
“那我到底该不该这么做？”薛雷抓了抓头发，“我被你搞糊涂了。”
“当然该。我是在提醒你，你必须明白自己做的事情意味着什么，别拿这个决定去找法萨尔公主谈判，试图让她说服姑姑，给予咱们全国传教许可。那只会起到反效果，咱们最好偷偷摸摸地进行。库里提卡卡塔就是个做实验的好地方，我非常赞同你的计划。但对其他领地能不能做同样的事，希望你一定要先跟我商量，再做决定。”

波赛思抬起眸子，凝视着他的表情，“我明白，出于各种原因，你都不会完全信任我。但，我说，薛雷，在洛库尔兰，相对其他贵族来说，至少我们家……应该是最可信的吧？如果你将来有了更加可以信赖的帮手，我不介意退出当前的角色，为她腾出位置。”
