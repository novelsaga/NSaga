# 第88章 孵出来的与孵不出来的

坐在已经被榨干精力半裸睡去的苏琳身边，薛雷于停止的时间中，陷入了短暂的纠结。
恩赐的两个选项里，已习得技能是对他目前来说有很高优先级的“魂交”，毕竟今后每晚睡觉都要过来这边度假，早点提升能力搭一个舒服的家，就可以在这边宅着顺便玩苏琳。
但另一个新技能选项，是他此前没见过的，名叫“图腾”。
预告性质的简短介绍中，这个技能能让指定物体扩散女神的庇护。
从神识难得波动了一些的口吻中判断，这个技能对传教有很大帮助。
好钢要用在刀刃上，舒适度往后再拖拖的决定并不难做。
纠结的是，他觉得苏琳身上的已习得技能有点太多了。
把这个也激活之后，她就有七种技能在身，不考虑那个奇葩的运气属性带来的影响的话，今后每一个技能每次升级的概率会降低到七分之一。
再加上见到之后还要优先选择的“洞察”和“同乐”，手气……不对，精气不顺的时候，急着提升等级的技能可能要四、五天才能见面一次。
这么想，薇尔思暗示过后续可以增加苏琳升级次数的“同乐”，优先度就得调到更高。
【其实你不必太过担心这些问题。】
啊？我这不是担心升级效率吗？
【薛雷，你的运气已经达到了这个世界常人无法触及的高度。那个属性其实是对你帮助最大的。你难道没有发现，你的锚每次出现的选项概率并不是你想象的那样吗？】
薛雷考虑了几秒。之前技能还太少，以后多了谁知道呢。
【我可以这么告诉你，你的运气，其实在影响着你每次选择时候看到的选项。不仅是锚，其他女孩得到恩赐的时候也是一样。只不过就像我说的，你对锚的影响力远比其他人要大，运气的效果当然也是如此。】
所以呢？
【所以你只要按照运气的指引选择，未来就不会后悔。你所希望的，未必就是你真正需要的。相信命运吧。】
薛雷考虑了一会儿，选择了“图腾”。
不相信薇尔思，那他还当个屁的代行者。
详细的技能说明，如约而至。
【图腾，可以指定一个具备女神象征意义的物体，令其拥有神力加持，可以对周边一定范围内的信徒造成与神力相关的正面影响。影响效果的类型取决于你持有的神权，而影响范围，取决于你选定物体的性质。选定物体必须为人工造物，价值、大小和醒目程度都会对范围进行提升。每次升级可以进一步提升影响范围的倍率。图腾技能发挥作用需要你或者锚的任何一个存在于影响范围内。同一时间只能指定一个图腾。】
哦哦，这不就是为了那个造价不菲的女神像量身打造的吗？
可激活之后呢？难道以后都不能离开这里了吗？
想到选项上的光芒，他又追问了一下后续技能的情况。
后续技能叫做“神恩”，“图腾”5级后就可以进入技能池，从预告上看，貌似是个弱化版永久性的“图腾”。
看来要是想让女神像的效果留存，就得在离开前拿到“神恩”才行。
一天两次的进度，忽然觉得好少啊……
中断“魂交”后，薛雷回到正常睡眠中，再睁开眼恢复意识，已经是六月最后一天的早晨。
黄金麦穗商会的各种手续已经办妥，为了尽快进入正常运营，文拉尔在商贸区挂出了薪水优厚的招聘
只要长得好看，经验啊专业知识啊什么的，向女神祈祷然后去取精就是。
总觉得这思路微妙地跑偏了一些，但暂时没有更好的办法，薛雷也就没再多说，按文拉尔的要求，每天负责进行最终面试。
作为教宗出席了一下第一批传教士的礼拜堂例会后，他回到圣殿，以苏琳为媒介，对着高大壮观的女神像，激活了新拿到的“图腾”。
肉眼看不出什么明显的变化，薛雷自己也感觉不到什么。
但激活的一瞬间，周围忙碌的那些有了粗浅信仰的帮工，就一起向女神像这边看了一眼，神情稍有变化。
他立刻回房，去找一早起来就沉迷魔法研究的拉雅。
她作为契约了的信徒，果然受影响更大。
按她颇为笨拙的描述，就在图腾激活的那一刻，她感觉到了被温暖的力量包围着，精神更加振奋，心情更好，也更有自信了。
这样的效果听起来还是有些玄乎，他干脆问了问神识。
【当前你只完整持有我的神权，那么图腾的主要影响就是信徒的生产技能收获提升，繁育后代的能力提升。】
可以壮阳？那这宣传放出去岂不是无敌了？
【效果不到药物那么明显，但的确会提升男信徒的生育力。】
这个神像应该够大了吧？图腾的范围大概是多少？
【按当前的技能等级，这个神像差不多能覆盖半径210里左右。】
哦，才210……等等！210里？210000米？
【是的，你这次选择的图腾非常完美，覆盖范围很大，悠远古林的一部分都被囊括进来。你在这个区域内保持活动，信徒们就会受到我的庇佑。锚的各方面消耗会略微减少。祭祀的效果也会略微提升。今后请尽量保持图腾范围的覆盖，这对你有好处。】
薛雷盘算了一下，目前人鱼之冠对原神教稍有了解的居民已经相当多，采购一批小礼品，在圣殿大门口这个广阔的院子里，距离女神像很近的地方召集几百个人每天举行祭祀，宣传上几天的效果应该不错。
文拉尔是行动派，知道这对女神有莫大的好处，立刻就拨款20金，命人去购买礼品顺便租下布告栏的位置安排宣传员。
薛雷去给仓库放置丰收额度的时间，她就已经把圣殿和礼拜堂雇佣的帮工全都召集到了一起，连着他们的家属，凑过来看热闹的路人，熙熙攘攘凑了大约三百多。
这里当然不能玩人鱼族群里那种祭祀方式，他向文拉尔和拉雅简单交代了一下，就利用剩余的几十斤丰收额度，进行了一次比较通常的祭祀。
收获的信仰值分量平平，加上后续过来围观的，将近四百的参与者，只提供了714点。
这714000经验都丢进魔晶石掌控里还不够挑战一次突破的，他就先加给了最近在灵魂世界比较需要的性技。
可能是很少给女孩子直接舔的缘故，他的性技·口水平惨不忍睹。
不过昨晚他舔了个够，算是尝到了让女人在自己唇舌间婉转呻吟扭动尖叫最后哭泣崩溃的美妙滋味，于是干脆地一发升到了等级7大师。
虽说他享受不到正常生命技能进阶额外给予的属性，但嘴里活动一下就发现，舌头还是实打实灵活了好几十倍的。
这会儿别说给樱桃梗打结，他都有信心配合牙齿在嘴里给樱桃切片剔核然后用梗摆盘。
也不知道神赐技能切换回性技·手的话会不会顺便提升做手工的能力。
不管怎样，几十万经验入账总是好的，而且被吸引来的人，说不定就有谁会对原神教感兴趣，成为入门的一般信徒。
万丈高楼平地起，打地基时不能急。
忙起来时间过得飞快，转眼上午就快过去，薛雷急忙抛下手头的事情，径直返回卧室，继续保持着压线放人肏苏琳的良好习惯。
新技能没有优先级，他这次选择了魂交到2。
试验了一下，果然能生成的摆设等级和总量都提高了一些，就先给她弄了个沙发，床上也换了舒适的席梦思。
电器貌似还没能力搞出来，他就给她弄了副扑克牌，丢了盒单人能玩的桌游。
测试发现，灵魂世界能生成的东西主要取决于他和苏琳的记忆，不需要太过清楚，只要是接触过有印象的，能力允许之后就能生成。
而且据神识所说，电器搞出来后，运转也不需要电。
只不过，那边是没能力联网的。
没有关系，打发时间的东西薛雷作为一个大龄单身宅男知道得太多了，等魂交技能高了，给苏琳生成一台电脑教她玩《文明》，那她估计就连吐槽的空都没了。
不知道是不是真正拿到了苏琳本体处女的原因，薛雷的心态也有了微妙的变化。
如果说肉体上结束处男意味着男人长大的一个门槛，那他昨晚可能算是突破了心理上的童贞。
异世界的生活节奏当然不如地球上的现代文明那么快速，加班如文拉尔，也会在晚饭前准时切换为休息状态，神礼日这天，大部分行业也都不会开工。
生活舒适度很高，但工程进度让人很焦躁。
这天下午，朵莱米托一条在水鬼海湾工作的同胞传话，说她当时产下的十个卵竟然孵化了八个，现在八条小人鱼绕着她的圈喊妈妈，她短时间内是没空过来看他了。
想象了一下单亲母亲应付八个娃的惨状，薛雷当然不会有什么意见，还让那条人鱼捎了一些营养品过去。
关于那八条小人鱼到底算不算他女儿这个问题，他一直纠结到了晚上睡觉前。
看拉雅和文拉尔都已经沐浴更衣完毕等待侍奉，他才抛开那不好给出答案的哲学问题，准备交今天份的粮。
可这时，塔蜜尔拍了拍门，“薛雷，你睡下了吗？”
手已经放在拉雅小巧乳房上的他尴尬地停下动作，扭头问：“怎么了？”
“如果你还没有休息，我建议你去浴池看一看古莎。”
“啊？”薛雷一骨碌下床，马上把宽大的袍子套在了身上，“那边出事了？”
晋升大魅魔至少需要三天，7月1号晚上古莎能出来就是女神保佑，这会儿能出什么事？
早产了？
“有侵入者进去了。”
“什么？”他吃了一惊，扭头叮嘱她们两个躺好先聊会儿，就穿上鞋飞奔出去。
古莎是他契约心腹中毫无疑问的最强者，在蕾莉亚不一定能随时召唤出来，塔蜜尔三态总和只有5的情况下，他可不敢让这个即将到手的大魅魔有任何闪失。
“你先不用急，对方没有恶意。而且……气息很熟悉。你进去的话，应该不会有冲突。”
薛雷本来听了这话放心了几分，结果一到门口，就看到微醺的米奥拉正盯着门缝呲牙，背后的尾巴都竖起来在半空炸着毛。
塔蜜尔过去拉开大猫，给了他一个让他放心进去的眼神。
有这位女法师在外面，只要来的不是格莱娜，他就没什么好怕。
他深呼吸几次，打开门，从缝里探头看了看。
巨大的茧还在水池中央静静的搁置，四面八方的丝线都已经固化，非常坚硬。
一个穿着黑袍的高挑女郎站在茧的旁边，伸出一只手搭在上面。
“你是谁？”薛雷急忙走进去。
那女郎扭过头，看到是他，抬手拉下兜帽，一阵黑雾散开，露出了夏拉那张妖艳到令人心悸的脸。
“你来了？”他彻底放下心来，这只大魅魔力量被扣得很厉害，而且她不是塔蜜尔的对手，又有血誓，完全不需要担忧。
夏拉的神情颇为感慨，“我真没想到，那位女神殿下有如此伟力，古莎一个刚成年的小魅魔，也谈不上勤奋，竟然这就不依靠仪式，自行结茧了。”
薛雷已经很熟练，马上微笑着说：“所以我一早就劝你，信仰女神吧，女神并不鄙夷黑暗。因为黑暗本就是光明的另一面。”
“以后再谈吧，我这次是找借口过来的，虚空通道还在等我，我没时间多待。”
夏拉的手一直没有离开那个茧子，他这才注意到，似乎有隐隐约约的黑色粉尘在灌注，“你这是在干什么？”
“兑现承诺。我在把我的知识，趁古莎正在重新塑造自我，简单点灌输给她。”夏拉盯着他，提醒说，“你也要记住你的承诺，五十年内，都不准让古莎来打我位子的主意。”
“放心，我说了，我们对你的位子不感兴趣。”
“那么，还有另一个承诺。”她扶着茧张开嘴，伸出灵活的舌头在空中勾了一勾，“过来吧，我的美餐。”
“你不是急着回去吗？”他脱下长袍下到池子里，走到蹲下后只露出个头在水面上的大魅魔身前，“不怕赶不及？”
“你别忍着，就不会赶不及。”她自信满满地扯掉湿漉漉的罩袍，舌头托着阴茎送入口中，双眼亮起了亢奋的紫光。
几分钟后，知识的灌输结束，夏拉站起来，离开浴池，幻术变化的黑袍再次出现在她身上。
她擦掉唇角残留的粘液，感受着恢复了一点的力量，扭头妩媚一笑，说：“多谢款待，下次见。”
这大魅魔三态总和依然只有10，其中淫欲占据了绝对比例，也不知道征服到契约还需要多久。
薛雷靠着茧洗了洗还有点发酸的鸡巴，点点头，“下次见。”
“对了，”她走到门口，又回过头，“古莎可能不会太快出来，你别心急，也别打扰她。”
“不会太快？”
“是的，不会太快。”夏拉的眼神显得有些复杂，像是在……嫉妒一样，“我的血脉和你的女神，也许共同打造了一个超乎想象的怪物。如果她不是我的女儿，刚才我就已经出手破坏了。”
“那她需要多久才能出来？”
“不知道。那要看她的素质如今到底有多强了。”她拉起兜帽，手放在了门把上。
“等等，你还答应我在苍穹魔堡传教来着，我怎么感觉没什么效果啊。”想起还有一个承诺，薛雷忍不住问道。
夏拉身体一僵，没回头，语速飞快地说：“我交代给手下了，至于进度，我很忙没空一直盯着。我们魔族不太擅长这种事，你过后有空亲自过来传播吧。再见。”
哗啦，她打开门，大步冲了出去。
〖她肯定是忘了。〗
“嗯，心虚得太明显了。”
既然知道了古莎的实力可能有超乎想象的飞跃，薛雷不得不更加慎重，把米奥拉的工作调整为全职守护浴池，不再需要去工地搬砖，薪水不变，每天还给报销酒钱。
然后，他又去拜托了一下欣蒂和塔蜜尔，但考虑到阵营和立场，他没明说古莎的情况，只说在茧里的时候会非常虚弱，万一出什么状况实在是太危险了。
为了不让欣蒂太过抵触，他还答应会让部下采购一套《光与暗之战》的基础包卡牌，让她在这儿跟大猫喝着酒消遣。
毕竟是一起战斗过的同伴，欣蒂很豪爽地答应下来，而塔蜜尔也点点头，柔声说：“既然今后还要一起旅行很久，同伴的托付，我当然也会尽可能出力。”
他松了口气，总算安心回房交公粮去了。
他本来以为古莎三天出不来，兴许第四天晚上就见面了。
可没想到，一直到7月5号早晨起床，晴天很多的青空之月转眼就要过去四分之一，古莎还在那个大茧子里面稳如泰山，不见任何动静。
今天正好是暗礼日，他只能希望他的小魅魔今晚会顺利出产。
这几天薛雷的生活和上辈子工作时候一样稳定，上午去筹备例行的祭祀，往库房堆积丰收额度生成的货物，午休后先为商会应聘的姑娘们面试，接着帮文拉尔处理教务，给这位今后教派内的实权人物刷刷好感，晚上带着拉雅逛街放松一下免得研究魔法研究得脑子烧掉，之后左拥右抱交公粮。
只不过睡觉后发动魂交，他暂时没再去折腾苏琳次次重置的处女膜，只是让她口交领了恩赐，就陪她玩会儿游戏，聊聊天，实现一下当初想象中和她恋爱后会发生的种种幻想。
当然，那时候的幻想，还纯情到不包括口交这一项。
几天的恩赐下来，按照职能的不同，薛雷给拉雅和文拉尔做了不一样的规划，拉雅主要提升在战斗侧，而文拉尔在属性足够自保的情况下，全都投入在了能辅助教务的技能上。
从属性和技能上看，拉雅应该已经是个非常强悍的白魔法师，可她整天对着老师教的知识，依旧愁眉苦脸好像一个面临高考的中游学生。
文拉尔这边收获则比较丰富，管理技能从起步新手火箭蹿升到等级7大师，而本来底子就很优秀的创作·文学更是在三次恩赐的联合努力下艰难突破了超凡的门槛，即使等级暂时只有1，未来也不可限量。
这样一来，教典之外的各种东西，他就可以放心交给文拉尔去撰写了。
