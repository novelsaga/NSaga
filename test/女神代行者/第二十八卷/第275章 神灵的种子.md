# 第275章 神灵的种子

火礼日的清晨，薛雷刚睁开眼睛，就从神识那里得到了一个还算不错的消息。
之前播撒在希拉米特姐妹体内的神灵之种，已经吸纳到了足够的能量，顺利从灵魂中分娩，转化为了可以给“锚”提升的恩赐。
这次，他选择了“附着”五级后激活的后续新技能——延伸。
此前拿到“附着”后他还一直没有机会拿琳琳实际使用，技能等级也没高到穆琳可以替代的程度，让他对这套冷门体系一直抱持着比较怀疑的态度。
他肯使用恩赐机会点高，纯粹是因为好奇，和恩赐数量比较富裕，他有闲情逸致浪费一些罢了。
但没想到，配合上后续的新技能之后，竟然产生了非常绝妙的化学反应。
“延伸”这个技能的效果并不复杂。
激活这个技能的前提，是“附着”正在使用中。
激活后，薛雷可以凭意识从被苏琳“附着”的物体上，生成他任意肢体的局部。
那部分为完全精神力产物，不可见，只能被感知察觉，会和原本对应的部分共享体验。
持续时间和使用次数，都和“附着”一致。
刚听到神识的解释时薛雷还有点懵，但简单的举例之后，他就立刻明白了这玩意的具体用法。
比如，他可以让苏琳“附着”在某扇窗户上，“延伸”出自己的眼睛获得里面的视野。
再比如，他可以让苏琳“附着”在某位女士的坐垫上，“延伸”出双手和老二直接来个远程透明奸。
根本就是套在战略游戏中非常无敌，放进黄色游戏中也极其带感的技能组合。
如果换成早些时候，薛雷可能还要思量一番，把苏琳禁锢在某种物体里对生性活泼的她是不是有些过分。
但现在，他已经在脑海里构思各种稀奇古怪的玩法了。
比较遗憾的是这个技能只能反馈各种感受回来，不能同步传输圣精过去。
否则，他就开辟了一条“梦魇”之外的远程注入新道路——开打的时候直接把苏琳附着到对方内裤上，延伸小兄弟往里插，当场削属性给敌人变废物。
当然，以他如今的性技水平，从女性敌人内裤里延伸出一根手指，也够他让对方丧失战斗力了。
还可以把苏琳“附着”在床上，和喜欢的姑娘躺在上面做爱时，他主体的分身从前面来，“延伸”的精神阴茎从后面入，独自一人就能实现三明治夹心双穴齐开，想想就让他胯下的肉龙蠢蠢欲动。
随着法瑟的病退，新神恩圣女莱希雅公主的到来，不管贵族和王室哪一边笑到最后，薛雷的创神会在洛库尔兰的前景已经十分明朗，势不可挡。
所谓的软禁昨天一早就已经正式撤销，他去吃早餐前看了一眼窗外，保护此处安全的，已经都换回了希拉米特家的直属卫兵。
薛雷的心情很好，觉得自己立了大功的紫虫更高兴。
花妖精本来就不怎么需要进食，摄入的能量比只偶尔出来吸两口火晶石的露狄还少，性价比极高。

“不止。”
薛雷仍然望着窗外，看着蓝天上一列列飞过的鸟，“我还想拆掉那些，玷污了生育这个伟大天职的笼子。没有人该替她们决定关于繁衍的规则，她们才是要做母亲的人。”

波赛思单膝跪地，手抚胸膛，深深低下了头。
倒并不是她被智慧女神的光芒所震慑，而是她在看到之后忽然警醒，自己此刻的身份。
“感谢您的降临与祝福，慈爱的女神。”
艾露迪特合上硬皮书，满意一笑，“我就在这里休息吧。我喜欢这种到处都是书的地方。明天早晨我去找你，在那之前，你只要告诉我，我该教些什么就好。”
薛雷坐到桌边，拿出收集的各种教材，“最重要的是这些基础管理技能，虽说有女神这个向心力，大家应该比较团结，但咱们选的人都是在家族中底子不好的，需要靠能力累积必要的威望。此外，和帮助教会发展有关的知识，有多少教多少，重要的是类别，而不是深度，哪怕让她们所有本领都只会个入门也好。”
艾露迪特微微蹙眉，蕴含着无穷知识的眸子散发出晶莹的光，“薛雷，对上位者来说，精通要比广博更加重要啊。”
“但对女神的恩赐来说，广撒种，短期内更有效率。”
薛雷无意对艾露迪特隐瞒什么，“你来洒下神灵的种子，我来帮助其中的好苗子迅速成长，然后，让她们在洛库尔兰各个地方，顶起属于咱们的天空。”
艾露迪特想了想，微笑着招了招手。
架子上一本本书被柔和的光芒包裹，缓缓飞来，落在桌上，堆叠成整齐的三摞。
“好，那，我就从今晚开始备课吧。”
