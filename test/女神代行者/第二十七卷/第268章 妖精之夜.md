# 第268章 妖精之夜

就像是意识到了挡在前方的盾短时间内无法被攻破，那柄单手斧微微晃动了一下，突然向后撤出，从原路飞了回去，消失在之后来袭的魔炮光辉之中。
根本来不及思考太多，薛雷在这一刻忘记了穆琳变身而成的魔龙其实是丰产女神用魔龙石和圣精积蓄的力量塑造出来的东西，并非真正存在于这个世界的实体生命。
他只是根据自己想要保护穆琳的本能，在伤口外亮起的强光中，瞬间收回了穆琳。
于是，遮蔽保护着他的庞大魔龙之躯瞬间消失，只留下两块耗光了能量的苍白石头。
薛雷这一次赌赢了。
闪耀着多彩虹光的盾牌并没有消失，依然忠实地保护着他。
黑铁要塞同时发射的三道魔炮，带着本该将这个广场至少一半土地翻搅起来的威力，轰击而来。
不可思议的奇迹，也随之发生在众人眼前。
如同轻烟撞上强风，粉雪撒入温泉，细小的浮尘被流水冲刷……魔炮的光，还没轰在虹光之盾上，就消散在空气中，变得无影无踪。
以那闪耀的盾牌为中心，半径数百米内的魔晶石，都在急速失去储蓄的能量。
广场下安置的防御阵眨眼间失去了效果，就连照明的灯，也飞快变得黯淡，连飞虫都难以再吸引过去。
虹光之盾变得越发明亮，接着，所有人都听到了一个声音。
“再对吾的伴侣不敬，待吾复苏，便把你们尽数屠灭！一个不留！”
那声音像是发自一个年幼的女孩，显得尖细而稚嫩。
可她的语调满含愤怒依然沉稳，说话的方式更是古朴到有些异常，好似才开始学习当前的通用语一样。
但没有人一个怀疑那句话的效力，包括已经面无血色的法瑟。
不仅是因为那小小的盾牌挡下了惊天动地的一击。
更因为他们此刻，看到了从未出现过的可怕奇迹。
被那面盾牌吸收掉能量的魔晶石，竟然失去了所有的光泽，甚至不再透明，也感应不到任何魔力的波动。
就像是，变成了普普通通的石头。
在这个世界当下智慧生命心中的常识里，这景象绝对不可能发生。
魔晶石能以稀少的储量成为露比哈特所有文明最关键的基石，根本原因，就在于其令人安心的特性。
每一块魔晶石的大小、纯度都不一样，蕴含的魔力和释放的效率，自然也都不同。
但它们有一个堪称神明恩赐的共同点，那就是只要不激活进入魔力释放状态，就会从周围吸取元素的能量逐渐补充。
地礼日的时候，地元素魔晶石还会得到额外的充能速度。
不管对魔晶石进行怎样的操作，不管从中压榨出多少魔力，这个特性都不会变。
可现在，只要是对魔晶石有基础使用能力的人都能意识到，虹光之盾吸收过的魔晶石，真的“死”了。
薛雷知道这是蕊妮莎的手笔。
他不知道蕊妮莎是怎么做到的，但他还没有蠢到这就暴露另一个秘密的地步。
从心音中确定穆琳没事，只是消耗了两大块魔龙石和一大截圣精能量后，他就开口帮忙掩饰说：“法瑟，看到丰产女神的愤怒了吗？你们难道想招来真正的神罚，才肯认错？”
法瑟的脸上还是没有半分血色，但她的语调非常稳定，“教宗先生，你既然毫发无伤，那以你的能力，应该知道，有些事情并不是我的主意。有些戏，我不得不演。”
“看来我就算问你为什么，也没有任何意义。”
“没错。我不知道，也不会问。”
薛雷冷笑，“觉得这样我就该原谅你？”
法瑟拿起剑，横在自己的脖子上，“你如果需要我这样道歉，我只希望你能考虑一下后果。”
薛雷哼了一声，“真的都是他的主意？”
“我也想看到真正的你。”
法瑟盯着他的眼睛，依然没有掩饰脸上的嫌恶神情，“我看到了。我并不后悔。这证明，我之前源自直觉的不安，没有出错。法萨尔太年轻了，她以往的成功，也来得太过容易。”
“我就是一个弱小的人类。单打独斗，你随便一个卫兵都能杀了我。”
薛雷平静地说，“但我是丰产女神的代行者。对我的冒犯，就是对女神的冒犯。你应该庆幸，薇尔思不是那些伪神。”
法瑟垂下眼帘，轻轻叹了口气，“我此刻，的确很庆幸。我可以用我的一切发誓，今后绝不会再有任何……以我的意志为主导的，对你不利的行为。”
“这话的意思，就是如果还有，我可以直接向另一位追责，对吗？”
她保持了沉默，从表情看，大概是承认了这个说法。
薛雷板起脸，大声说：“那就请殿下转告那个藏头缩尾的混蛋，提醒一下他，我已经退让了够多，我爱好和平，但不意味着，有人打我我不还手。他如果做好了承担一切后果的准备，就尽管来吧。”
广场陷入了短暂的沉寂，只剩下被石化了部分身体的卫兵在绝望地呻吟。
几缕浅绿色的光随风出现，在法瑟身边凝成一个漂亮的环。
塔兰从里面走了出来，表情肃杀。她环视了一圈，确认薛雷没事之后，看向法瑟，“告诉我，发生了什么？”
法瑟挥挥手，让周围的卫兵散开远离身边，压低声音，简短地做了一个说明。
塔兰的目光渐渐变的错愕。她紧紧抓着法杖，纤瘦的身体竟愤怒到微微发抖。
“你知不知道，你做的决定有多么愚蠢？”
法瑟挺直脊背，平静地回答：“愚蠢，但是必要。我不想让任何一点不安定的火星，降落在这片和平太久的干草堆上。”
塔兰的唇角微微颤动了两下。她转过身，化作一片碧色的风，吹到薛雷的身前。
“我不知情。”
薛雷笑了笑，“但如果我死了，你也不会做什么，对吧？”
塔兰没有否认，“你没死。”
“所以，你打算做什么？”
“我会暂时离开，去见神王莉娜。”塔兰一贯平静的脸上，显出了鲜明的怒色，“以这种方式绕过规则使用世界意志的力量，必须受到惩罚。”
“怎样的惩罚呢？”
她思考了几秒，“我不知道。我的记忆中，这样的事，还是第一次发生。”
“希望你能给我带回来好消息。”薛雷毫不客气地说，“否则，我就只能自己想办法了。”
塔兰带着沉重的表情，缓缓点了点头。接着，她抬起法杖，整个人化作无数纤细的光丝，飞入无垠的夜空，消失不见。
薛雷心里还有很多关于守护者的问号。但他考虑了一下，决定不问。
他现在更想通过另一种方式来了解那一切——比如，制造一个绝对不会对他撒谎的守护者。
神恩圣女的事情尘埃落定，没有让法瑟完全如愿，格兰诺瓦的密谋彻底失败，薛雷今后只会更加警惕。
他略一思忖，认为不需要再跟广场上的人有任何交流，就干脆地发动技能，消失在他们面前，回到了希拉米特家的庄园。
如先前同伴通知的一样，这边的气氛，也相当紧绷。
放置着女神画像图腾的那个房间外，一队全副武装的士兵，正在法萨尔的身后，跟面前的蕾莉亚对峙。
菲尔汀带着一队骑士远远站着，但只是观望，没有要插手的意思。
蕾莉亚的身后，是已经放出了所有同族的蒂尔宁。
棕色的辉光构成若隐若现的法阵，闪耀的鹿角都已经做好了施放的准备。
让薛雷有点意外的是，布莉妲拿着一把防身匕首守在门口，手还在发抖，而在她的头顶，紫虫正满脸愤怒地盘旋飞舞，身上的花裙变成了方便刺杀的款式。
砰。
薛雷用力关上房门，提醒对峙的双方，他，平安回来了。
这时，他才注意到，房门两侧还站着许多希拉米特家的卫兵，从阵型来看，他们竟然是在听从布莉妲的指挥。
看到薛雷出现，布莉妲像是被抽走了筋，长长松了一口气，靠着门边的墙瘫软坐下。
紫虫毫不掩饰自己的惊喜，径直飞过来，先是嫌弃地把缠绕在身上的泉仙子让渡给他，为他戴上收纳着炎魔露狄的魔晶石坠饰，嘴里喋喋不休语无伦次地说了一大串。
“我就知道主人没事。我可是最棒的幸运妖精。那些身份尊贵的小婊子嘴上就没一句实话。你那个护身精灵到底有没有用啊？她保护你什么啦？主人不是去王宫做客了吗？为什么这么晚才回来啊？我们在这儿等你，还有公主过来捣乱，不让我们等。真是讨厌得要命。今天明明是人家跟你亲亲热热地好日子，为什么会有这么多人过来捣乱啊？我的幸运难道都让给主人了吗？”
薛雷根本不知道怎么接茬，索性直接看向蕾莉亚，“我回来了。这边怎么了？”
蕾莉亚仍盯着法萨尔的咽喉，轻声说：“没什么，我看很晚了，想在这儿等你。法萨尔带人来说不用等了。然后，莫名其妙就变成了这种局面。”
“咱们后面那些卫兵是怎么来的？”他真正的疑惑是这个。
“布莉妲带来的。”
蕾莉亚想了想，小声补充，“应该是波赛思她们不方便直接出面，用非血亲的私生女，万一出了什么岔子，还可以甩掉一部分责任。”
薛雷若有所思地点了点头，往前走出两步，看向法萨尔，“这么晚了，公主殿下还没休息？等着我，是有什么要紧事吗？怎么在琳琳身边布置那么多士兵，她很胆小的，你没发现她都快吓哭了吗？”
琳琳站在四个高大的卫兵中间，确实已经是一幅要哭出来的样子。
法萨尔的目光从上到下，从头到脚仔仔细细扫了一遍。她的表情控制能力远不如法瑟，立刻就变得十分迷茫，甚至掺杂着掩饰不住的恐惧。
“公主殿下？”他放大音量，又提醒了一声。
法萨尔醒觉过来，浑身一颤，马上收起了握着的武器，很勉强地挤出一个笑，说：“我收到王都的指示，今天教宗先生在那边册封了一位神恩圣女。我特地率人过来迎接，以王室成员的身份，对你表示感谢。”
紫虫把嘴巴快要撇到脸蛋上，气哼哼地说：“看吧，是不是没有一句实话。”
蕾莉亚缓缓举起手里的剑，“你确实应该感谢，感谢我的薛雷平安无事回来了。”
法萨尔勉力维持着脸上的笑容，“王宫是全国最安全的地方，你不需要如此担心。要不是紫月症的威胁还在，我也不会调动这么多人来这儿等待。毕竟，教宗先生是洛库尔兰的仰仗和希望。”
她很快往后退去，“那么，我就不打扰你们休息了。再会，祝你们有个美好的夜晚。”
薛雷捏住鼻梁揉了揉眉心。他发现，他和波赛思很可能过分低估了对手。
事情发展到这个地步，很明显，对面的那些人里，除了苏琳自己，已经没谁还相信护身精灵的性命和他直接挂钩。
是我演技太差吗？他挠了挠头，转身安抚担忧的女伴们。
一番交流之后，薛雷比之前更加意外。
布莉妲的确是十分担心去找波赛思商量，才拿到了一支卫队，匆匆赶来。
但没想到的是，提醒她让她感到不安的人，是傍晚才来到庄园中的瑞娜。
王宫中的侍女可以没有任何战斗能力，可以不太擅长份内的工作慢慢学习，甚至可以偷懒摸鱼只要能混过去，但唯独不能没有对环境变化的敏感性。
瑞娜算是敏感性极强的小天才，大概正是这个原因，她姐姐才不惜冒险抓住机会让她离开。
她过来和大家认识，被比较热情的接纳后，就有些担心地提出了自己的忧虑。
当然，她小小一个侍女，不可能知道法瑟背后的密谋究竟源自什么。
她只是依靠直觉的敏感性，认为薛雷被国王这么热切地拉拢，恐怕会有军方的人找他的麻烦。
所以之后布莉妲一发现蕾莉亚在女神画像房间外跟法萨尔对峙，就忙不迭跑去找姐姐求助，希望神通广大的姐姐和母亲能化解紧绷的局面。
涉及到守护者格兰诺瓦的秘密不适合让太多人知道，薛雷回去后，找借口支开不肯走的紫虫，等房间里只剩下蒂尔宁和蕾莉亚，才把路上隐瞒没讲的部分，简略告诉了她们。
蒂尔宁听完，差点一蹄子踏裂地砖。
一想到那把凶狠的斧子当时距离心爱的鹿爸爸脑袋只有不到十厘米的距离，她就浑身发冷，四条腿和双手一起哆嗦。
魔物出身的麦伦愤怒中还难以避免地混杂着对守护者的恐惧，而蕾莉亚看似平静的双眼中，就只剩下了森冷的杀意。
旧恨新仇，此刻，尽在心头。
“亲爱的，你累吗？”
“我还好。”
薛雷知道她的意思，过去抱住蒂尔宁，拥吻片刻，变出一把最新等级“丰收”才能生产的顶级地属性魔能植物——星落荧藓，笑着喂到她嘴边。
蒂尔宁舔了一下，尝尝味道，跟着毫不犹豫低下头，柔软的嘴唇和舌腹飞快刮蹭着他的掌心，把里面的宝物吃得干干净净。
“怎么样，味道如何？”
她眨眨眼，轻易被他用美食转移开了情绪，“唔，加些沙拉酱可能会更好，直接吃有点涩嘴。但……魔能好充沛啊，这东西……纳加斯古石塔一辈子也见不到几次呢。吃到一回，足够小部落凑一起欢庆好几天了。”
“要休息了，别吃太多。就，稍微加点特殊的沙拉酱吧。”他握拳，打开，变出了新一把星落荧藓，配着一份圣精。
“嗯。”蒂尔宁捧住他的手，温顺地舔舐到嘴里，咀嚼，咽下，帮助她的鹿爸爸，把宝贵的恩赐用来给锚升级。
之后，她开门回自己房间休息，跟乐滋滋飞进去的紫虫走了个擦肩。
“主人主人主人，我已经哄睡布莉妲了哟。我说啦，我唱安眠曲很棒的。虽然……可能没有你那么棒，但够用了呀。”
“很好。”薛雷抬起手，如她所愿，揉了揉她的头，顺便召回了被嫌弃的有些伤心的小沐，“紫虫，我发现，幸运妖精可能真的有效。”
“那～～当然。”紫虫得意洋洋地抬起小脸，叉腰在空中转了个圈。
“但我还是有点担心你的身体，咱们，就先来试试，用手指，好吗？”
她歪着头想了想，笑眯眯地飞过来抱住了他的手，“好呀。我相信不管怎么样，主人都一定能让我非常舒服的。”
那么，被突发事件打断了的实验，在此重启。
薛雷让小沐分出一部分身体，包裹住他的小拇指，但调整了几次，发现怎么摆放其余手指，都感觉非常别扭。
“用这根吧。”紫虫伸出两只小小白白的赤脚，左右夹住他的中指，灵巧地上下摩擦。
薛雷带着几分感慨，心想，这只花妖精，真是把纯洁和淫乱毫无违和地融在了一起。她扬着无邪的笑脸，双脚却做着比酒吧舞娘还要撩人的事。
“这根不会太大吗？”他比划了一下，有些担心地问。
“大……才能测试出来呀。”
紫虫咬唇窃笑，站在他掌心弯腰，往指尖上妖媚地舔了一下，“我以这种形态，能装下主人的这根东西，那我变大一点之后，主人就不需要担心把我撑裂了呀。”
“让她试，让她试。”也不知道是看热闹不嫌事儿大，还是之前被嫌弃记了仇，小沐在旁边荡漾着波纹鼓动起来。
“相信她吧，亲爱的。”蕾莉亚从后面拥抱住他，轻轻吻上他的耳朵，“你这都是为了我，不管有什么后果，我都来和你一起承担。”
“不要说得跟人家会被插死一样。”
紫虫不高兴地撅起了嘴，“你们真正应该担心的，是我的身体太棒，主人今晚之后就被我迷住，到时候失宠了，可不要生我的气哟。”
小沐一边把身体的辅助部分从小拇指转移到中指的指甲上，一边毫不客气地嘲讽，“你该不会不知道，咱家主人还有个魅魔情妇在外面办事没回来吧？而且，那还是魅魔公主哦。那个等级的怪物，只要愿意，恐怕能让教宗的手指都射出精液来。”
“没那么夸张。”薛雷打断了两个小东西一开头就总是没完没了的斗嘴，“紫虫，我的手怎么摆放，你会比较方便一点？”
