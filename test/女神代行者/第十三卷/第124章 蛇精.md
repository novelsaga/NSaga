# 第124章 蛇精

按说，非群居的魔物对他的帮助极其有限，信仰传播效率不高。
但，薛雷不太舍得错过这个目标。
首先，他一路看着动物世界的内容飞过来，难得见到一个人性化的生物出现，当然想要上去交流一下。
其次，她长得可实在不差，秀气可爱的小圆脸，微有波浪的长卷发，即使鳞皮裙遮挡着身材，光靠相貌也是很标准的美少女。

那个锥子脸妖婆画进本子里都骚得不行，眼前这个，可是实打实的青春美少女蛇精，稍微化化妆就是个妖娆版桥本环奈，他岂能不鸡动。
那只米娅抬起头，注意到了飞至湖面上方的古莎，但很快她就把视线放回到龙卷持续攻击的鳄头兽身上，双掌在胸前相对，准备进行下一段咏唱。
鳄头兽的防御能力相当强悍，一个完整的中阶下位风系黑魔法全部吃下，看上去依旧生龙活虎，怒吼一声用前爪踏扁身上已经死亡的猎物脖子，把柔软的蟒躯从身上拽下。
薛雷本来以为这么一个长着鳄鱼头名字叫鳄头兽的怪物应该有鳄鱼一样的游泳能力，可没想到那家伙看到对岸那只米娅的身影，只闻了闻水面，就沿着湖边撒开腿跑了起来。
“这玩意不会游泳的？”

“而且这俩都不能飞，你有翅膀，怎么会有威胁。”他推测说，“我看，湖里应该还有什么东西存在。”
“那雌兽肚子里的鱼是怎么抓的？我看这个雄的也不会做钓鱼竿啊……”
“看，它不是不敢下水。”薛雷指着已经爬入水中躲避连续风刃的鳄头兽，“它应该是不敢从刚才那个深度穿过去。湖底估计躲着什么厉害的家伙。”

从她瞬魔法的施展轻松程度来看，魔力相当丰沛，技巧也很娴熟，缺的，大概是实战经验。

很明显，拉法特蟒这样的蛇类魔物移动并不是强项，进化种带有人类半身之后，不仅体型小了一圈，速度也会变得更慢。
这个距离下，她甚至来不及放出下一次杀招，就要被张开的血盆大口咬住。
“古莎，上。”
再看下去，可爱小蛇精就要变成怪物的大粪了。

还有几分迷茫——她想不通为什么那只魅魔要胸前带着“食物”飞过来，对她示威吗？

没什么智能的魔兽眼睛里仅剩下那只袭击自己的米娅，此刻就想用牙齿撕开她柔软的身体，把富含魔力的血肉吞噬成自己生长强大的养料。

同理，150力量和200的掰手腕，胜率其实就相当于15力量对上20的。
所以，体质三属性本身就是短板的古莎，并没有选择在正面和鳄头兽那样身强力壮的怪物顶牛。

他站到岸边，双手比划，聚精会神地生成一个又一个漩涡，趁机熟悉这权能的操作方式。

就像在玩什么小游戏一样。

不出所料，听到那些和生育有关的福利，繁衍为天职的进化种少女立刻双眼放光，细长的蛇信子冲着薛雷的胯下连连摆动。
但偷偷瞄了一眼古莎，卓娜丝胆怯地缩回尾巴中央，手指头交错拨弄，小声说：“古莎小姐能找到这么好的依靠，真令我羡慕呢。”
“那都是因为信仰。对薇尔思殿下的信赖，可以祈求到伟大的祝福。”薛雷走近蹲下，温暖的掌心轻柔抚摸着她冰凉光滑的蛇鳞，“你们魔物进化种的争端和矛盾，不都是因为生育力这种繁殖资源的稀缺吗？如果生育力非常充裕，可以满足你们各族群的进化种，是不是就可以不再为了地盘争斗？”
卓娜丝倒是很诚实，小声说：“那样进化种之间的确不需要再争斗，原始种倒是还会继续有摩擦。毕竟原始种都还不太会思考嘛，要在进化蜕变中吸收长辈传来的知识才行。不然都只能和同胞这样交流。”
说到最后，她举例子一样抿紧小嘴用鼻子发出一串轻轻的嘶，大概就是她们作为拉法特蟒的时候交流的方式。
苍穹魔堡位于高空，本来就一直吹着风，随着日头偏斜，气温变化，风速也渐渐变大，不远处的树林，已经像是到了夜店的少女一样摇晃。
薛雷想了想，古林灵鹿是这片悬空陆地上最大的魔物族群，指望一举拿下不太现实，既然已经跟卓娜丝认识，二鸟在林不如一鸟在手，还是先把可爱的小米娅得手再说。
虽说不是群居对他的掌控力帮助不大，但她长得可爱啊。
“你的巢穴离这里远吗？快傍晚了，我送你回去吧。”
“嗯。可以的话，请跟我多说说薇尔思殿下的事情，好吗？”
“好，路上我慢慢跟你讲。”
讲完还是不太懂的话，我还可以实际操作演示一下，让你知道这被祝福的生育力有多么澎湃。
“蝙蝠兔子鱼老虎，现在又是蛇，雷哥，我托你帮我养的那只狗……没被你怎么样吧？”
你今晚是打算带狗尾巴肛塞汪汪叫着挨肏吗？
“开个玩笑啦，小气。”
我是很认真问的，你有兴趣吗？
“呃……试试，也不是不行。”
看苏琳拿起书挡住脸，满眼小后悔的样子，薛雷笑了笑，没再理她。
他盘算着，今后要是遇到原始种外形类似蝎子的魔物娘，一定要带来跟卓娜丝凑一对儿，结成双飞组合——蛇蝎美人。
这要做到游戏里，得是个隐藏成就吧？
