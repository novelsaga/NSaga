# 第214章 蔓延的狂暴

2021年11月5日
薛雷很庆幸，灵魂世界里的他比外面还勇猛能干，精力充沛源源不断。
否则，古莎和穆琳这组搭档，他还真有可能应付不来。
而且，3号的狂欢过去，4号古莎来后，竟然又拉着穆琳一起上阵，比以前大方了不知道多少倍。
看她每次大乱斗完心满意足的样子，以后有跟穆琳同场竞技的机会，她八成不会错过。
不过这俩凑到一起对他来说其实还有一个好处。
他真的精神疲倦没什么兴趣继续的时候，她俩还能穿戴上各种玩具彼此满足。
只是他在旁边看着听着，往往坚持不了多久，就得挺着硬邦邦的鸡巴摇头叹息丢开手柄，忍不住过去加入。
古莎这次离开后不久，“神恩”的时间到了。
薛雷过去看了看苏琳的情况。
她没什么好转，正呆呆地躺在床上，保持着之前被留下时候的姿势，嘴角干涸的精斑都没去擦。
他叹了口气，叫来穆琳，让她帮忙照顾着点儿，旋即，结束技能，回到了肉体之中。
蒂尔宁负责白天，这会儿紫色的月亮已经爬到了高空，守在外面的，自然是他亲爱的蕾莉亚。
娴静状态下的光精灵女郎，有种令人不忍心凑近打破那画面的精致美感。
可惜，蕾莉亚并不是在休息，而是在戒备。
一听到门响，她就立刻转过头，旋即，都等不及绕过椅背，就那么抬脚一踩，飞身跳过来，扑到了薛雷的怀里。
她仰头送上满含思念的唇瓣，贴在一起轻柔地彼此摩挲。
片刻之后，还不舍得分开的他们，听到了旁边传来的提醒：“我说，你们平常也是这么旁若无人的吗？”
薛雷按住蕾莉亚想要挪开的头，继续吻了一会儿，才轻轻舔掉她柔软唇瓣上的唾液，看了过去，“还有什么比这更好的表达思念的方式吗？”
“我还在旁边看着呢。”波赛思双手抱肘，沉甸甸的乳房堆积在胸口并不富裕的空间里，把法师袍撑起奇妙的弧度。
“波丝娜被你咬奶头疼得嗷嗷叫的时候，我在做的事比接吻更容易接受？”
波赛思翻了个白眼，“好吧，下次你把我可爱的妹妹屁股肏开花的时候，我会记住不看。”
知道她这个时间还不休息，亲自等在这儿绝对不会是因为下面痒了发骚，薛雷搂着蕾莉亚的腰，问：“外面情况怎么样了？你没来打断我，是不是说明还不错？”
“我这儿可没什么好消息。”波赛思过去擦了擦蕾莉亚踩过一脚的椅子，坐下，双手搁在椅背上，下巴枕着胳膊，看起来很疲倦的样子，“不然我早就在房间里泡我的花瓣澡了。”
“很糟糕？”
“很糟糕。但还没糟糕到让我决定把你弄出来的地步。”
蕾莉亚摸出之前寄放在她这儿的火晶石，挂在薛雷身上，让露狄继续担当火系宝可梦，轻声说：“她来过几次，确实忍住了，没有叫你。”
“我给蒂尔宁留的东西吃完了吗？”薛雷比较关心自己的伙伴，“要是吃完了，有什么话咱们就去她房间说。她隔三差五要放同族出来吃喝，负担很大。”
“还没。”蕾莉亚摇摇头，“她说看不到你没什么食欲，吃得很少。”
波赛思看到薛雷脸上浮现的焦虑，乖觉地说：“走吧，咱们去那可爱的小鹿房间里谈。”
等蒂尔宁含着他的指头舔掉最后一点酱汁，心满意足靠在他怀里轻声哼唧的时候，他已经大致明白了这段时间洛库尔兰的情况。
的确算不上严重。
紫月症惹出的乱子如果放在水鬼海湾那样的地方，大概能让所有波及的城市焦头烂额。
但这里是洛库尔兰，辉煌平原上最强大的人类王国，洛卡拉联邦的主心骨，一度被认为有希望诞生另一位女皇的地方。
每晚发生的损失和骚乱，都能得到有效地控制。
但现在的问题是，要到什么时候才算完？
“神恩”持续的五天里，因为紫月症发作而被捕的囚犯已经超过五百。
从发作数量的变化来看，还在稳定增加。
最让人不安的是，最后这两天，紫月症患者中已经出现了此前绝对没接触过月光教的个体。
他们的共同点，是接触过紫月症发病状态下的患者。
“也就是说，看到紫月症带来的强化表现，并因此而动心的人，就有可能被那股神秘的力量侵染。”波赛思喝下口水，润润喉咙，继续说，“这可是很大的隐患。每次抓捕患者，都需要出动很多士兵，要是这样传播下去，我想陛下只能考虑让大家今后都在家里放个大笼子，一到晚上就自觉点进去了。”
薛雷抚摸着蒂尔宁鹿角之间柔顺的银发，“那，之前我治疗的那些人，有复发的吗？”
“有……一个。”波赛思的口吻显得有些阴郁，“就是我妹妹，波丝娜。”
“她这么快就复发了？”
“是的，而且比之前更强了一些，还能召唤月仆来帮她，给家里惹来不少麻烦。妈妈很生气，觉得她简直是希拉米特家的耻辱。”
薛雷若有所思地点点头，“其他人短时间没事就好，波丝娜我可以帮忙控制住。被抓的囚犯都关押起来了？”
波赛思托着腮，小幅度地摇了摇头，“昨晚宫廷法师们确认患者有可能让非教派人士也发病之后，今天中午，各地抓捕的囚犯，都公开处决了其中的大部分。”
薛雷吃了一惊，“处决了？”
她嗯了一声，“我说，这不是理所当然的处理方式吗？既然它已经有了传染病的苗头，你的宝贵药剂又不可能全面铺开解决，那么，杀掉源头，都不需要国王或元帅下令的吧。”
薛雷揉了揉眉心，“好吧，薇尔思殿下的信仰还需要时间铺开，这的确是个好法子。”
“暂时你只能在希拉玛卡塔传教。”她显得有些遗憾，“母亲争取过了，国王陛下那边没什么意见，但元帅大人不同意。她认为这次紫月症就是宗教暗中传播导致的恶果，在根除之前，对贵教也会谨慎观望。”
国王没意见，元帅不同意？
薛雷皱起眉，这才意识到，他此前光顾着了解洛库尔兰各大贵族的资料，还没仔细调查过这边的王室到底是怎么个结构。
他匆忙在心里找神识补了补课，这才了解到，露比哈特这个女性力量和数量都占优势的世界，人类的国度大部分依然是由国王作为元首，女王的存在，寥寥无几。
就连北方和洛卡拉联邦历史上纠葛颇多的人类帝国露比修特，尽管出现过的几代女皇均是出色的精英，当前这位年纪轻轻也不可小觑，但是，历史上仍以男性皇帝占多数。
因为人类的寿命在各种族中堪称倒数，女性的最佳生育年龄又极为短暂，孕期还特别长，对实力的影响特别大，就产生了一个对人类帝王影响极大的弱势——儿女过少。
实力的增加的确可以延长寿命保持青春，但实力越强的人类，生育能力就越弱，在正值青春的时候就修行到长寿，那几乎等同于绝后。
而且，学识、谋略、政务这些都需要领导者投入大量的时间，每一个名声在外的女王或女皇，都远远算不上强者，其中小半甚至可以说是英年早逝。
庞大的国家，繁多的贵族，不可能依靠单独一两个人的武勇来统御。
家族血脉的连结，也是上位阶层彼此拉拢的惯用手段。
边境的封臣兵强马壮，王室应该放心还是担心？
但如果那位封臣的第一继承人就是国王的外孙呢？
就算该担心还是要担心，起码能稍微放心一点。
所以，人类男性依靠强大的生育能力，享受着贵族体系中的大部分顶级门面待遇。
上到国王，下到某个小镇的镇长。
但门面待遇，说白了也就是拿来充门面，作为血统上的家长而已。
上限高，数量多，基础天赋优秀的女性，才算是这个世界人类文明的运转核心。
希拉米特有三个家主，唯一的男性不到三十岁已经有了三十多个孩子，每天除了打理领地事务就是兢兢业业奔波在妻妾的卧室间辛勤耕作。
而包括碧萨拉在内的两个女性家主，都选择了招赘夫婿，强化自身的路。
碧萨拉只有两个女儿，但大女儿波赛思年近三十，作为母亲的她依然保持着令人心动的美貌，足以说明她的强悍。
洛库尔兰王室情况也是类似。
这一代国王年纪已经不小，过了最佳生育年龄的他随时可能退位，交给最精壮帅气的儿子接棒。
很多仪式性的场合都由国王出席，贵族联席会议那样需要血脉连接紧密人物在场的状况，也是国王负责。
而大部分实际权力，则掌握在首席法师、大神官、元帅之类的王族杰出人才手中。
无疑，绝大多数为受女神眷顾的那个性别。
洛库尔兰重视武力的风气只要不改，掌握王室绝大部分直属部队，并能对封地下令的元帅，就是说话最有分量的人。
现任洛库尔兰元帅的，是法瑟·亚蒂罗森·库尔雷斯，当前国王库拉坦三世的妹妹，菲尔汀的姑姑。
“法瑟……是个很固执的人吗？”问出了最大的障碍，薛雷自然要继续打探一下。
只能在希拉玛卡塔传教并不是什么很重要的限制，但元帅大人的态度，可能直接影响洛库尔兰军队方面对他们的看法。
在这个几乎所有孩子都以成为厉害将军为荣的国度，被士兵们排斥，等于是被变相封杀。
波赛思歪着脑袋斟酌了一会儿，反问：“你关心这个干什么？那位可是大忙人，不太可能有时间听你传教。”
“我总要知道，出力解决紫月症，到底能不能改变她的主意。和希拉玛卡塔相比，整个洛库尔兰王国太大了，有没有希望，会影响我调遣人手的决定。我们现在可靠的传教士还不太多。”
疲倦的女法师不是很乐观地摇了摇头，“蒂罗森家跟我们家关系十分密切，菲尔汀公主跟法瑟在姑侄之外还有近似师徒的关系，而即使是这样……我妈妈找她商量你说的那个治愈方式的可行性，还是被很干脆地驳回了。”
“我说，你应该知道，带兵打仗的人眼里看什么都会觉得有阴谋。”她笑了笑，“这个世界充满了算计，那么，你们的教派，就没有可能是真正的罪魁祸首吗？当她这么想的时候，聪明人都知道不要再跟她讨论这件事。”
薛雷挤出一丝微笑，“你母亲是聪明人。”
波赛思起来伸了个懒腰，拉长的身段让曲线变得更加明显。
她就像是故意的一样，在最舒展的状态下扭了扭腰肢，美好的弧度就那么在法师袍中藏不住地晃荡，“那么，能跟我去看看我可怜的妹妹了吗？”
这一趟，去得比薛雷想象中要远。
不过蒂尔宁只要让他骑在背上就很开心，一路哼着歌儿，边走边摇晃脑袋，鹿角上镶嵌的珍贵饰品闪动着美妙的光芒，一副巴不得这段路走不完的模样。
关押波丝娜的地方当然已经不在庄园，而是在希拉玛卡塔中心那座巨大城堡的地牢。
不过，她毕竟还是碧萨拉的女儿，曾经备受重视的天才魔兽使，关押在这边的地牢主要是为了隔离开她能呼唤到的各种魔兽，并尽可能减少她对其他人的影响。
牢房里面专门重新作出的布置，实际上奢侈到一般贵族都无福享受的程度。
可惜，波丝娜被好几条符文锁链缠绕着固定在合金支架上，比被薛雷在灵魂世界性虐的时候还要夸张。
看外面相邻房间的装饰布置，这里还给她安排了女仆，只不过仅白天的自由行动时间可用，到晚上，就要为了安全一个不剩地撤走。
“她这不是挺安静的吗？”薛雷隔着带有雷属性融合晶石的栅栏看了一会儿，疑惑地问。
“月升之刻应该已经挣扎过了，你看她手腕的破皮。”波赛思的话中能听出对妹妹的关心。
但薛雷还察觉到，那语调里同样还有一丝属于虐待狂的兴奋。
根据上次同时玩弄姐妹俩的经验来猜测，波赛思大概是那种既能施虐又能受虐的类型。
下次干脆把她俩绑住吊起来玩一下好了，这种下作的乳量，很适合后手缚或龟甲缚。
看着波丝娜浑身上下的链子意淫了几秒，薛雷看向蕾莉亚。
蕾莉亚点点头，意思是大可放心，有她在。
“解开她的嘴，我试试看能不能让她恢复理智。”
不想让狱卒看到自家人狼狈的样子，那一大串钥匙就在波赛思的手指上挂着。
而附带的魔力波动检测，她也是允许通过的几人之一。
牢门打开，很快，除了堵住嘴巴还有防止自残功能的口枷就被拆下。
双眼闪动着紫红色的光芒，波丝娜盯着薛雷喘息，唾液从破了皮的嘴唇之间垂下。
她身上只穿着最底线的遮羞布料，不过一路走来所有负责的人员都是女性，倒也问题不大。
“哈啊……哈啊……薛雷，异神的……使徒……必须……消灭……”
一看到薛雷的脸，她的四肢又开始发力，链条在支架上发出丁零当啷的声音。
他懒得多废话，抬手捏住波丝娜的脸颊，想了想，又拿起口枷，把存货圣精变出一发抹在上面，塞了回去，按住。
很快，她双眼的妖异光芒就迅速黯淡下去。
“薛雷……”掏出口枷后，波丝娜轻唤了一声，眼里的恨意迅速消失得干干净净，但是，倒也看不出什么感激的意思，“果然……又是你。”
“当然是我。”薛雷盘算了一下，随手可以喂的圣精在苏琳里面还存着43发，虽说不少就是在波丝娜身上存出来的，但都用给她，可有点不太舍得。
薇尔思，为什么在她身上效果的持续时间就那么少？
【波丝娜应该是被对方施加了比较强的共鸣。她对月亮女神已经没有多少信仰，但她非常渴望力量。最糟糕的是，她极度排斥对我的信仰。所以单纯靠吸收圣精里的赐福来对抗，对她的效果远不如那些普通的非信者。】
那该怎么办？
我马上要回人鱼之冠帮忙讨伐海盗，可不想被贵族拖在这儿。
【稍等……我需要从庞大的知识中寻找对应的答案。】
好吧，连搜索引擎都卡了。
薛雷挠挠头，适时露出为难的神色。
波丝娜倒是不怎么在意的样子，有气无力地说：“我没救了，对吗？”
波赛思抬起她的下巴，仔细修剪过的眉毛不顾形象地聚拢在一起，“你如果一直这样下去，的确没救了。但是，我说，咱们不是还在想办法吗？别忘了，人能爬过鹰都飞不过去的山。”
“可我已经是山脚下的烂泥了。”波丝娜扯动嘴角，奇妙的是，她的眼睛里闪动着象是从什么重压中释放出来的轻松，“姐姐，紫月症帮我找回了力量，我才发现……我原来已经不想要了。”
“嗯？”波赛思疑惑地看着她，“你在说什么？”
“我说啊，我……感觉好累。姐姐，原来有力量，比没有力量累那么多。”波丝娜低下头，“就让我腐烂在这儿吧。没救，也不是什么坏事。”
波赛思一把抓住她的头发，揪起她的脸，强迫她和自己对视。
“我说，你以为，流着希拉米特家血液的你，有资格随便决定自己的未来吗？”
波丝娜的嘴唇颤动着，呻吟般说：“我已经是个被送给宗教头目玩弄的废人性奴隶了，这样的未来……你还不满意吗？我已经……不再是那个有可能掌控魔兽部队的……天才了，姐姐……还是不能放过我吗？”
波赛思凑近她的耳朵，说话的热气喘息一样喷吐出丝丝的痒，“不能。你也是妈妈的女儿，就是烂掉，也得烂在妈妈指定的地方。给我乖乖爬起来，如果，你不想月仆今后跟在别的婊子背后摇尾巴。”
波丝娜抖了一下。
似乎对她来说，从修行开始就一直陪伴着她的那些魔兽，比家人更加珍贵。
姐妹交流的同时，薛雷也等到了神识给出的解决办法。
