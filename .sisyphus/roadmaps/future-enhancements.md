# 未来增强 & 待办事项

## 概述

非核心功能的长期路线图，包含稳健性、性能和基础设施改进。

## 稳健性增强

- [ ] **RPC 调用超时机制**

  - 状态: 暂缓 (目前本地 IPC 通信较快，死锁概率低)
  - 预计: 1-2天

- [ ] **Bridge 崩溃日志收集**
  - 需求: 捕获 Node.js 进程崩溃时的 stderr
  - 状态: 暂缓 (开发阶段可直接看终端输出)
  - 预计: 1天

- [ ] **配置热重载**
  - 需求: 修改 `.novelsaga.js` 后无需重启 LSP
  - 状态: 暂缓 (目前重启 LSP 速度很快)
  - 预计: 2天

## 性能优化
